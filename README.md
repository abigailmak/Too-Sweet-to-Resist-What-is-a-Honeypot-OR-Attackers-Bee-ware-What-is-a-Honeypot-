# Too-Sweet-to-Resist-What-is-a-Honeypot-OR-Attackers-Bee-ware-What-is-a-Honeypot-

Using Digital Ocean, I deployed a server on the cloud with no protections. This server had no firewall or counter attack policies. It was just a default server with no protections other than a password. I was able to gather data about approximately 22,000 attacks. This data ranged from the attack methods to the attacker geolocations and reputations. 

Using the free platform, T-Pot, I was able to better analyze the mass amounts of data I received from just 24hrs. When analyzing the collected data, I was able to apply what I learned in my co-op as a SOC Analyst to interpret and hypothesis the what, why, when, where, nd how. Cyberattacks happen more often than most people realize, its a 24/7 business. When you have the opportunity to understand how your attackers will act, you have the opportunity to better defend yourself.
